# 🧭 2. Git Flow - адаптированный

Kлассический Git Flow слишком тяжёлый. **Oблегчённый Git Flow**, идеально подходит для Nuxt‑проектов.

## 🔹 Основные ветки
- `main` — стабильная версия (деплой)
- `develop` — рабочая ветка, куда вливаются все фичи

## 🔹 Временные ветки
- `feature/*` — работа над задачами
- `hotfix/*` — срочные исправления прямо в `main`
- `release/*` — подготовка релиза (опционально)
