## 1. Красивые и понятные имена для веток🎨
> Если ты работаешь над UI, SCSS‑архитектурой, компонентами, навигацией, типографикой — поэтому имена должны быть:
- короткие
- читаемые
- отражающие задачу
- одинакового формата

---
### Варианты для проекта
*Формат:* `feature/<area>-<short-task>`

1. #### Для UI / SCSS
    1.1. feature/ui-sidebar-fixes
    1.2. feature/ui-footer-refactor
    1.3. feature/ui-nav-improvements
    1.4. feature/ui-typography-update
    1.5. feature/ui-layout-cleanup

2. #### Для компонентов
    2.1. feature/component-appnav
    2.2. feature/component-menu-card
    2.3. feature/component-footer

3. #### Для страниц
    3.1. feature/page-catalog
    3.2. feature/page-subcategories
    3.3. feature/page-home-hero

4. #### Для архитектуры
    4.1. feature/scss-tokens
    4.2. feature/scss-mixins
    4.3. feature/scss-global-refactor

```
